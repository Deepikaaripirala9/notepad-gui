# 📝 Notepad GUI (Java Swing)

A simple Notepad application built using Java Swing with basic text editing features.

## 🚀 Features
- Create new file
- Open existing file
- Save file
- Cut, Copy, Paste
- Scrollable text area

## 🛠️ Tech Stack
- Java
- Swing (GUI)

## ▶️ How to Run
1. Compile:
   javac Notepad.java

2. Run:
   java Notepad
